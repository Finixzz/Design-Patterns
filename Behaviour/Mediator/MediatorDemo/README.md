Simple .NET CORE WEB API project that demonstrates best practices using MediatR extension along with CQRS principle. Entity configurations are made with DataAnnotations.
DAL(Data Acces Layer) is present in separate .NET CORE class library for better project organization.

# Installing MediatR

You can install MediatR package via Nuget Package Manager with the following command:

```bash
Install-Package MediatR
```

Or via the .NET Core command-line interface:


```bash
dotnet add package MediatR
```
